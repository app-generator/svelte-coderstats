## Getting started

### Clone

```bash
$ git clone https://github.com/app-generator/svelte-coderstats.git
```

### Running the project

However you get the code, you can install dependencies and run the project in development mode with:

```bash
$ cd svelte-coderstats
$ npm install # or yarn
$ npm run dev
```
